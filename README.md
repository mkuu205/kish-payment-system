# Kish Payment System - Termux Edition

A complete M-Pesa STK Push payment system optimized for Termux testing.

## Features

- ✅ M-Pesa STK Push integration
- ✅ Real-time payment status
- ✅ PDF receipt generation
- ✅ Transaction history
- ✅ Beautiful responsive UI
- ✅ Dark mode support
- ✅ Mobile-first design

## Quick Start

### 1. Install Termux

Download Termux from F-Droid or Google Play Store.

### 2. Setup

```bash
# Clone or copy files to Termux
cd ~
mkdir kish-payment
# Copy all files to this directory
